Problem → Solution

Problem

Sidebar Diagnostics was my go-to, but I wanted a slim top-edge dock with less visual noise, better hot-plug resilience, and instant tweaks without relaunching.

Solution

TopMonitoring ships a 24px strip that hugs the top of the screen. Metrics reorder on the fly, labels rename instantly, and settings save to a portable folder so the setup follows you.

Developer journey

I started with zero coding knowledge and iterated nightly, learning WPF, XAML, and LibreHardwareMonitor APIs. Each release folded user feedback into a calmer UI, better defaults, and a tray workflow that respects creators recording or streaming.
