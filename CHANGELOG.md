# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-17

### Added
- Initial release of Top Monitoring
- Real-time hardware metrics overlay (CPU, GPU, RAM, VRAM, Disk, Network)
- FPS monitoring support
- Dark/Light theme toggle
- Customizable metric labels and ordering
- Autosave settings
- WPF-based Windows application

### Changed
- Updated README with local screenshot paths
- Removed external badges and links for cleaner documentation

### Fixed
- Various performance counter warnings for Windows compatibility

### Known Issues
- FPS accuracy depends on system capabilities
- Some GPU power metrics may show N/A on certain hardware
