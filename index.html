Windows Overlay Application

TopMonitoring

A lightweight Windows top-bar overlay to display real-time hardware metrics using LibreHardwareMonitor and WPF.

View Repository Learn More

Smart Monitoring Features

Minimalist Overlay

Always-on-top dock with transparent background support and MSI Afterburner-style layout for clean aesthetic.

Real-time Metrics

Monitor CPU, GPU, RAM, VRAM, Disk, and Network (Up/Down) with optimized update loops.

Highly Customizable

Custom colors (HEX support), opacity control, metric reordering, and label renaming at your fingertips.

Technical Specification

Requirements

  • Windows 10 / 11
  • .NET SDK 8 (Developer)

Developer Ready

  • Clean WPF Architecture
